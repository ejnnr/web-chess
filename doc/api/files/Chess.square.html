array2square()

array2square(array  $square) : integer

converts formats like array(1, 3) to an integer

Be careful: this function can return 0 or false, so use === to check the result

Parameters

array $square

the square to convert as an array ([$fileNumber, $rankNumber], both starting at 0)

Returns

integer —

an integer; bit 0 to 2 are the file number, bit 3 to 5 are the rank number and bit 6 and 7 are always zero (in LSB 0); returns false on failure
